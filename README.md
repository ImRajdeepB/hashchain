# Mina Snapp: Hashchain

Zero-knowledge proof of age using Hash chains.

https://asecuritysite.com/encryption/age

## Install dependencies

```sh
npm install
```

Set year of birth in [this line](./src/index.ts#L105)

## Run proof of age

```sh
npm run poa
```

## How to build

```sh
npm run build
```

## How to run tests

```sh
npm run test
npm run testw # watch mode
```

## How to run coverage

npm run coverage
